## タグのルール

- 既存タグを確認し、**表記ゆれを作らない**（例: ×「牛たん」→ ○「牛タン」、×「Nightreign」→ ○「ナイトレイン」）
- **英語タグは固有名詞の製品・サービス名のみ可**（Discord, iPhone等）。略語・一般概念は日本語に（例: GW → ゴールデンウィーク）
- **地名は市単位**を基本にする（籠原→熊谷、深谷市→深谷）
- 新規タグは「**今後2記事以上で使う見込みがあるか**」で判断。製品型番・一点物の固有名詞はタグにせず本文で表現する
- 子育て系タグは「**子育て**」に統一する
- 宿泊を伴う外出=「旅行」、日帰り=「おでかけ」の使い分け
- **タグは本文に実際に登場する内容のみ**。関連が薄い・本文に出てこない話題のタグは付けない（例: グルメの話が無い記事に「埼玉グルメ」を付けない）。迷うタグは依頼文に「（要照合）」と添えてPMに判断を委ねる

## 記事原稿の書き方

- H2見出し（`##`）の先頭に絵文字を1つ付ける（例: `## 🎬 …`）
- 本文中の画像は挿入位置にプレースホルダーで指定: **「【ここに〇〇を挿入】」**
- 動画の埋め込みは: **「【ここに動画を埋め込み：URL】」**
- 記事末尾は**応援メッセージで締める**
- 「※本記事にはAmazonアソシエイト…」の**免責注記は本文に書かない**（レイアウト側で一括表示）
- 記事末尾の並びは必ず: **本文 →（商品カードがあればカード）→ `---` → アフィリエイト定型文**
  （区切り線 `---` は定型文のすぐ上に置く）

### 表のルール

- 表のセルは**全角20文字以内を推奨**（長いとPC幅で末尾1文字だけ折り返される）
- 表・図の下の補足注記は `<p class="table-note">※…</p>` で書く（CSSは定義済み）

### アフィリエイト定型文（末尾に入れる全文）

```
このブログの商品リンクから飛んで、そのままAmazon・楽天・Yahoo!ショッピングで別の商品（日用品や消耗品など）を買ってもらっても、ブログへの応援になります😊リンク先の商品を買わなくてもOKです。もし使う機会があればぜひ活用してください✌️
```

## 投稿依頼文テンプレート

Chatで原稿を仕上げたら、以下の形式で投稿依頼文を作る。

```
■元原稿: （原稿本文 or 添付ファイル名）
■カテゴリ: （固定7種から1つ）
■タグ: （既存タグ優先・新規は2記事以上見込みのみ）
■投稿日: （例: 2026-07-03 12:00 JST）
■スラッグ: （例: example-article-2026）
■本文中の画像: （枚数と、どのプレースホルダーにどの画像かの対応）
■アイキャッチ(1200×675): （元画像ファイル名。16:9センタークロップ→リサイズ）
■アフィリエイト案内文: あり／なし（通常あり）
■商品カード: あり（商品名・リンク）／なし
デプロイまでお願いします。
```
